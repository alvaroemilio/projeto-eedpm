Técnico em Infomárica

O Curso Técnico em Informática é voltado para a formação de profissionais capacitados a lidar com as diversas tecnologias da informação, incluindo programação, redes, manutenção de sistemas e desenvolvimento de soluções computacionais. A seguir, apresento um resumo das disciplinas de cada módulo, abordando tanto os aspectos técnicos quanto as competências complementares.

MÓDULO I

Estatística Básica Técnicas de Comunicação e Treinamento Inglês Técnico Fundamentos de Hardware e Software Introdução à Informática Lógica de Programação

MÓDULO II

Sistemas Operacionais Redes de Computadores Linguagem de Programação Segurança de Sistemas Desenvolvimento WEB Manutenção de Computadores

MÓDULO III

Análise de Projetos de Sistemas Banco de Dados Empreendedorismo Programação Para Internet Programação Orientada à Objetos Projeto Integrador
